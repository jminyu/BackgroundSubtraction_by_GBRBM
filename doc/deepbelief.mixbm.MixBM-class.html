                           MixBM

This class can be used to draw samples from a Boltzmann machine whose energy is a weighted sum of the energies of two given RBMs, SemiRBMs or GaussianRBMs.

References:

  • Salakhutdinov, R. and Murray, I. (2008). On the Quantitative Analysis of Deep Belief Networks.

tune(self, weight)

source code 

Tunes the parameters of the mixed model so that its energy becomes

  • (1 - weight) * energy_one + weight * energy_two.
Parameters:
  • weight (real) - a weight between 0 and 1
