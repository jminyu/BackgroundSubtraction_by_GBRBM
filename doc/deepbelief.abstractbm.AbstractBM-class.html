Class AbstractBM

source code

Known Subclasses:

  • , gaussianrbm.GaussianRBM

Provides an interface and common functionality for latent-variable Boltzmann machines, such as contrastive divergence learning, Gibbs sampling and hybrid Monte Carlo sampling.

References:

  • Hinton, G. E. (2002). Training Products of Experts by Minimizing Contrastive Divergence. Neural Computation.
  • Neal, R. (1996). Bayesian Learning for Neural Networks. Springer Verlag.
